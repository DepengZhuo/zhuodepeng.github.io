⑩ 底层计算逻辑与估值模型说明

1. 企业价值 (EV) 还原

EV = 股票总市值 + 有息负债 - 账面现金。
逻辑：如果你全资收购一家公司，你需要买下所有股票，替他还清债务，但他账上的现金可以直接装进你的口袋。EV才是并购市场真正的衡量标尺。

2. 综合质量评分 (满分100)

ROE权重25分、ROIC权重25分；利润含金量(现金转换率)20分；造血能力(FCF率)20分；资产模式(CAPEX比率)10分。
逻辑：杜邦分析和波特五力的量化体现。低分公司说明要么处于周期底部血亏，要么商业模式极差，DCF模型对其完全失效。

3. 达摩达兰两阶段 DCF 折现模型

核心公式：$PV = \sum_{t=1}^{5} \frac{FCF \times (1+g_{1})^t}{(1+r)^t} + \frac{FCF_5 \times (1+g_{2})}{(r - g_{2}) \times (1+r)^5}$
我们将未来现金流分两段：前5年具备确定性增长，之后进入与通胀持平的永续增长。计算出现金流现值后，加上净现金（现金-负债），除以总股本，即为单股内在价值。

4. 安全边际 (MoS) 决策机制

公式：MoS = (内在价值 - 当前股价) / 当前股价。
逻辑：由格雷厄姆提出。即使我们算出的内在价值是绝对准确的，市场也可能发生黑天鹅。只有当股价跌到内在价值的 75折 到 85折 甚至更低时，才具备真正的防守反击能力。
