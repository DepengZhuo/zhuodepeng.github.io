⑩ 底层计算逻辑与说明

1. 数据源限制说明

基于开源免费协议，本平台可通过 API 自动获取公开市场的实时股价、流通股本、部分 TTM 财报数据（如 ROE、毛利率）。涉及到深度财务审计级别的数据（如有息负债、自由现金流）必须依托于专业终端查阅后手动填入。

2. 达摩达兰两阶段 DCF 折现模型

核心公式：$PV = \sum_{t=1}^{5} \frac{FCF \times (1+g_{1})^t}{(1+r)^t} + \frac{FCF_5 \times (1+g_{2})}{(r - g_{2}) \times (1+r)^5}$
